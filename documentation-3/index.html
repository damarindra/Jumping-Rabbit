Getting Started

Hi, This is documentation for purpose to guiding you to edit this game.

First you need Unity 3D. I'm using version 5.2.

Second, you need Android SDK, make sure you have setup it correctly on Unity. :)

Third, The Source Code

Fourth, Your assets for redesign this game.

Importing Project

For someone who experienced with Unity3D can skip this :D

How to :

  1. Extract the zip file from codecanyon, don't forget where you put the directory.
  2. Open Unity, then open file> point to your extracted directory.
  3. Wait for seconds, and taraaaa. You did it.

Setup Google Play Game Services

How to Setup : 

Key Ids :

  1. Open Assets/Resources/Scripts/Services.cs
  2. You can see any variable for ID, change it with your own
Make sure your setup on Google Play Console has been complete corectly
If you can't show Leaderboard or Achievement, Open Plugins / Android / MainLibProj / AndroidManifest.xml, make sure your APP ID correct. If wrong, change with your own

 

Setup Admob

How to Setup :

  1. Just go to Assets > Resources > Scripts > Services.cs
  2. Edit admobBannerID, admobInterstitialID, interstitialShowPerGame.
If you want to edit location banner, go to method BannerSetup() then change AdLocation.placewheredoyouwant

 

Change Assets

This step is depend on your skills.

Basics :

Just replace existing assets with yours. Then need some enchance. If object has collider, set the collider to the proporsion. I recommend to learn about Unity.

Information, all prefabs can be found at Assets/Resources/Prefabs

Character : 

  1. Click the character, then see at inspector.
  2. You can see SpriteSwitcher has member sprite with array size 3.
  3. First is for jump sprite, second is for fall sprite, third is for die sprite
  4. If character don't set the animation on awake.

Enemy Spike Buddy :

  1. Click the Spike Buddy, the see at inspector
  2. you can see SpriteSwitcher has member sprite and Animation on Awake set true.
  3. Member Sprite is for animation walk, change if you have yours.

Coin :

  1. Click on Coin, is just one coin.
  2. If you want to create Another Coin Group, just drag and drop the CoinGroup1 to hierarchy.
  3. Edit it.
  4. Last step, you can fount lastYPos is child of CoinGroup. set it to the top of coin Group. this will be used for spawn next object
